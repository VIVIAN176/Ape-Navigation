# 猿航 - 程序员导航利器
实用性：非常实用，我已经设置为自己的浏览器首页了


### 开发
```
yarn global add parcel-bundler
parcel src/index.html
```
### build
```
parcel build src/index.html --no-minify --public-url ./
yarn build
```

### 技术栈
- jQuery
- LocalStorage
- SVG Symbols
- @media

### 后续
可以用 Vue / React 重写这个项目，考虑添加各种炫酷的背景图片或动画
